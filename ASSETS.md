## NOTES IMPORTANTES

1. **Paramètres Midjourney recommandés** :
   - Version : `--v 6` (ou plus récent)
   - Pour les icônes : ajouter `--style raw` pour plus de contrôle
   - Pour les textures : ajouter `--tile` pour les patterns répétables

2. **Post-traitement** :
   - Retirer les fonds avec remove.bg si nécessaire
   - Redimensionner avec précision dans Figma/Photoshop
   - Exporter en PNG avec transparence pour les icônes

3. **Cohérence visuelle** :
   - Palette principale : Purple #6366F1, Cyan #06B6D4, Dark #1E1B4B
   - Style : Moderne, minimal, géométrique
   - Éviter : Dégradés trop flashy, ombres dures, styles datés
